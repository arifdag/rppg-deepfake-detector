Model Architecture

Technical details about the Visual + rPPG Fusion detection system

Visual Branch

EfficientNet-B2 backbone with Noisy Student pretraining extracts spatial features from face crops. Temporal attention pooling aggregates features across multiple frames.

EfficientNet-B2 (NS) 224x224 input 8 frames/video

rPPG Branch

Remote photoplethysmography extracts subtle blood flow patterns from facial video. The POS (Plane Orthogonal to Skin) method analyzes physiological signals that are difficult to fake.

POS Algorithm FFT Features 42-240 BPM Band

Fusion Head

Features from both branches are concatenated and processed through a multi-layer perceptron. Focal loss handles class imbalance during training.

MLP Fusion Focal Loss Dropout 0.4
